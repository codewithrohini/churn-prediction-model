# 🛡️ ChurnGuard AI: Customer Retention Dashboard

An interactive Business Intelligence (BI) tool built with **Streamlit** and **Python** to predict and analyze bank customer churn.

## 🚀 Features
- **Executive Dashboard:** High-level KPIs and demographic trends.
- **Customer Deep-Dive:** Filterable database to find high-risk segments.
- **AI Predictor:** Real-time churn probability scoring using Machine Learning.

## 🛠️ Tech Stack
- **Frontend:** Streamlit
- **Visuals:** Plotly Express
- **ML Model:** Scikit-Learn (Random Forest)
- **Data:** Pandas / Numpy

## 📦 Installation
1. Clone the repo: ' https://github.com/codewithrohini/churn-prediction-model.git '
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
